SessionConfiguration

data class SessionConfiguration(var keyStrategy: SessionKeyStrategy = SessionKeyStrategy.ChatUser, var storage: SessionStorage = InMemorySessionStorage(), var managerFactory: SessionManagerFactory = SessionManagerFactory { bot, cfg -> DefaultSessionManager(bot, cfg.storage, cfg.keyStrategy, bot.config.loggerFactory) })(source)

Configuration block for the SessionManager subsystem.

The subsystem is always-on with sensible defaults; calling sessions { } in BotConfiguration is only required to override keyStrategy, storage, or managerFactory. Auto-tracking is predicate-driven — the interceptor only writes when a session has been opened (via bot.sessions.of(update) / bot.sessions.get(...)), so untouched bots pay no storage cost.
