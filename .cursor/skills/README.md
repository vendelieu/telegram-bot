# Telegram Bot Skills

Skills are organized by audience:

## library-dev/

For **library contributors** extending the telegram-bot library:

- `add-telegram-api-action` — Add new API methods
- `add-telegram-api-tests` — Add API action tests
- `add-telegram-update-field` — Add new update types
- `add-ktnip-collector` — Add KSP collectors
- `add-ktnip-ksp-tests` — Add KSP processor tests
- `add-ktnip-parameter-strategy` — Add parameter resolution strategies

## bot-user/

For **bot developers** building bots with the library:

- `build-inline-keyboard` — Inline keyboards
- `build-reply-keyboard` — Reply keyboards
- `setup-bot-handlers` — Command, input, common, update handlers
- `create-input-chain` — Multi-step input chains
- `handle-callback-query` — Callback query handling
- `build-wizard-flow` — Wizard flows
- `add-wizard-handler` — Configure @WizardHandler
- `add-wizard-step` — Implement WizardStep
